# java-lotto
로또 미션 진행을 위한 저장소

# 구현해야할 기능
1. 입력
    1. 구입금액 입력
        1. 0 이상 정수인지 확인
    2. 당첨번호 입력
        1. 쉼표로 구분
        2. 개수가 6개인지 확인
            1. 중복제거
        3. 1 ~ 45 사이의 정수인지 확인

2. 로또번호 생성
    1. 중복하지 않는 정수 6개 무작위 생성
    2. 1 ~ 45 사이의 정수 인지 확인
    3. 로또 개수만큼 생성

3. 추첨
    1. 당첨번호와 로또번호를 비교
        1. 숫자가 3개 ~ 6개 일치하는 로또의 개수 카운트
    2. 수익률 계산
